VARIAVEIS

Imagine variáveis como uma gaveta nos quais você pode armazenar certas informações (meias, cuecas, cinto...), estas coisas são chamadas de valores das variaveis. Cada gaveta deve ter seu próprio nome, pelo qual poderemos indicá-lo claramente.

Temos somos livres quando se trata de escolher esses nomes, mas devemos sempre escolhe-los com base no qe vamos armazenar. Por exemplo. se vamos criar uma variavel o gaveta para guardar meias então esse deve ser o nome da nossa variavel.

Em outras linguagens de programação, uma variável deve ser declarada antes do uso, e JavaScript não é exceção. Declarar uma variável é simplesmente "reservar" o nome da variável. Dessa forma, informamos ao programa que na parte posterior da execução, usaremos esse nome para nos referir a nossa gaveta, a fim de recuperar um valor dele, ou guardar um valor nele.

Os nomes de variáveis em JavaScript podem ser qualquer sequência de caracteres. Mas há um conjunto de palavras reservadas que não podem ser usadas para nomear variáveis, funções ou qualquer outra coisa. Elas são partes integrais da linguagem e recebem significados que não podem ser alterados.

Em JavaScript, os nomes das variáveis podem ser qualquer sequência de letras minúsculas e maiúsculas, dígitos, caracteres de sublinhado e cifrões, mas não devem começar com um dígito.
NOTA: O interpretador JavaScript distingue entre letras maiúsculas e minúsculas, também em nomes de variáveis, nomes como: teste,Teste, ou TESTE serão tratados como diferentes.

Declarar variáveis

Para as declarações, usamos as palavras reservadas ou palavras-chave var ou let para variáveis e const para constantes.

Execute o Codigo abaixo

    
        var height;
        console.log(height);

A cima declaramos uma variavel com a palavra reservada ou chave var e o height é o nome da variavel. A declaração, como outras instruções JavaScript, deve terminar com um ponto e vírgula
Resultado: 

    
        console.log(height);  //  ->  undefined (indefenido)
        console.log(weight);  //  ->  Uncaught  ReferenceError:  weight  is  not  defined (weight não foi definido)

Como ainda não colocamos nada lá, o interpretador conhece esta variável, mas ela ainda não tem valor – o valor é indefinido.
Na próxima linha, tentamos imprimir o conteúdo da variável weight ... que esquecemos de declarar. Desta vez, veremos Erro de referência. O interpretador JavaScript, nos informou que não conhece uma variável com esse nome (portanto, a variável em si é indefinida).

Var & Let

A palavra-chave var vem da sintaxe original do JavaScript e da palavra-chave let foi introduzido muito mais tarde. Portanto, você encontrará var mais em programas mais antigos. Atualmente, é altamente recomendável usar a palavra let.

Uma das diferenças básicas no uso de var e let é que let nos impede de declarar outra variável com o mesmo nome (um erro é gerado). Usar var permite que você redeclare uma variável, o que pode potencialmente levar a erros na execução do programa. 

    
        var height;
        var height;
        console.log(height); // -> undefined
    

    
        let height;
        let height; // -> Uncaught SyntaxError: Identifier 'height' has already been declared
        console.log(height);

Inicializando variáveis

A inicialização pode ser feita junto com a declaração ou separadamente como um comando independente. É importante inserir o primeiro valor na variável antes de tentar lê-la, modificá-la ou exibi-la. 

    
        let  height  =  180;
        let  anotherHeight  =  height;
        let  weight;
        console.log(height);  //  ->  180
        console.log(anotherHeight);  //  ->  180
        weight  =  70;    
        console.log(weight);  //  ->  70

A propósito, preste atenção em uma coisa. Se você especificar um nome de variável em console.log, o interpretador o reconhecerá e exibirá seu valor. Se você colocar o mesmo nome entre aspas, ele será tratado como texto simples e exibido como tal. 

    
        let  height  =  180;
        console.log(height);  //  ->  180
        console.log("height");  //  ->  height

Declarações e modo estrito

Veja o exemplo abaixo

    
        height  =  180;
        console.log(height);  //  ->  180

À primeira vista, você pode ver que esquecemos de declarar a variável height. O JavaScript permitia tal negligência. Mas infelizmente às vezes pode levar a situações ambíguas e potencialmente errôneas. 

    
        "use  strict";
        height  =  180;  //  ->  Uncaught  ReferenceError:  height  is  not  defined
        console.log(height);

Alterando valores de variáveis

Variáveis, como o próprio nome sugere, podem armazenar dados que irão variar. As alterações são feitas atribuindo um novo valor à variável, que sobrescreve o anterior 

    
        let  steps  =  100;
        console.log(steps);  //  ->  100
        steps  =  120;  //  ->  120
        console.log(steps);
        steps  =  steps  +  200;
        console.log(steps);  //  ->  320
